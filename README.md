# Development of a New Traversal and Performance Optimizations for Verlet Cluster Lists in AutoPas

This repository contains the measurement data and generated figures used in the Bachelor’s thesis:

**Development of a New Traversal and Performance Optimizations for Verlet Cluster Lists in AutoPas**

Bachelor’s Thesis in Informatics  
School of Computation, Information and Technology  
Technical University of Munich (TUM)

Author: **Stanislaw Jerzy Grzegórski**

```

│
```
